## Remarks

The caster's shield is enchanted based on its available enhancement bonus.  e.g.If the shield can be enchanted to +4 but already has a +1 enchantment, plusThreeEnchantment is applied.

See the SacredWeapon blueprint for example usage.

All enchantments default to the corresponding TemporaryEnhancement* blueprint.
