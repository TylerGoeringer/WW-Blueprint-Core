## Remarks

This is useful as part of an AddInitiatorSkillRollTrigger with StatType.CheckIntimidate and [`IsDemoralizeAction`](../BlueprintCore.Conditions.New/IsDemoralizeAction.md) to apply additional effects to the Demoralize target. If this not used then the context target will be the character using Demoralize.
