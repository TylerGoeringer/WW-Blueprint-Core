## Remarks

These log events print to the same output as Wrath's game log events. They can be viewed using [RemoteConsole](https://github.com/OwlcatOpenSource/RemoteConsole/releases) or by appending `logging` to the executable startup arguments. If you are not using the console, logs print to `AppData\LocalLow\Owlcat Games\Pathfinder Wrath Of The Righteous\GameLog*.txt`.
