## Remarks

Functions are split into three groups:

* **ContextRankConfigs** – Base class which creates a config with a specific base value. You can only use a single type, so you should only call these functions once for a config.
* **[`ProgressionExtensions`](./ProgressionExtensions.md)** – Extension class which exposes different progressions. Like the base value, you can only have a single progression type so you should only call one of these functions.
* **[`CommonExtensions`](./CommonExtensions.md)** – Extension class which exposes common configurable values.

See [ContextRankConfig](https://github.com/TylerGoeringer/OwlcatModdingWiki/wiki/ContextRankConfig) on the wiki for more details about the component and [ContextRankConfig Calculator](https://docs.google.com/spreadsheets/d/11nQdJ7DFzS73gwR9xk3gsKbyGgtDM51yNoMv7nNYnPw/edit?usp=sharing) for help determining which progression to use.

Create a rank based on Strength with a bonus value of 2 and a max value of 30:

```csharp
var rankConfig = ContextRankConfigs.BaseStat(StatType.Strength).Max(30).Add(2);
```
