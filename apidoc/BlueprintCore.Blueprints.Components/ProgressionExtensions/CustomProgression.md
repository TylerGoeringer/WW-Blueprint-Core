# ProgressionExtensions.CustomProgression method

Implements Custom

```csharp
public static ContextRankConfig CustomProgression(this ContextRankConfig config, 
    params ProgressionEntry[] entries)
```

## Remarks

Entries must be provided in ascending order by their BaseValue.

The result is the ProgressionValue of the first entry where the config's BaseValue is less than or equal to the entry's BaseValue. If the config's BaseValue is greater than all entry BaseValues, the last entry's ProgressionValue is returned.

```csharp
ContextRankConfigs.CharacterLevel()
    .Custom(
        new ProgressionEntry(5, 1),
        new ProgressionEntry(10, 2),
        new ProgressionEntry(13, 4),
        new ProgressionEntry(18, 6));
```

* **Levels 1-5** – `Result = 1`
* **Levels 6-10** – `Result = 2`
* **Levels 11-13** – `Result = 4`
* **Levels 14+** – `Result = 6`
